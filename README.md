# ODEIO
#### Just a simple boilerplate to create projects with Grunt and Sass

## Using  

### Install
```bash
npm install
```

### Usage
```bash
grunt default
```

### Tasks

- jshint
- concat
- uglify
- sass
- cssmin
- imagemin
- http-server
- watch
